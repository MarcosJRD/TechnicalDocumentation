Introduction

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

What are the things required for HTML programming?

The basic stuff

  • All computers will come with some of the basic tools necessary to write HTML and CSS, and to edit images. First of all, you'll need a text editor to write your HTML and CSS files. If you have a PC and use Windows, you can use Notepad, the most basic text editor on your computer.
Uses of HTML
  • It is used for basic layout creating or designing the Web page
  • Withouth HTML, the World Wide Web will not exist
  • It allows embedding text, image, multimedia (audio / video) and links to other documents and the Web pages.
  • It provides a means to create structured document by using paragraph, character formatting, links and lists.
  • It can embed scripts such as CSS, JavaScript, which after the behavior and design of the Web page
HTML Advantages
  • It is highly flexible and user friendly.
  • It is an open technology that supports almost all Web Browser and platforms like MS-Windows, Macintosh, U
  • It is efficient and reliable. You can create the Web Page in order to advertise and promote products and services.
  • It is easily understandable and does not require long time training.
  • It provides search engine compatible to Web sites.
HTML Disadvantages
  • It is complex to design attractive Web pages only using HTML. So, other languages are used for additional programming
  • It is Dificult to develop a complete Web site by using only HTML. Web developments tool like Dreamweaver, Foundations are used
  • It is cannot be used to develop a dynamic Web page
  • There is no any complete acceptable standard of HTML
  • There are many incompatibilities of HTML
JavaScript and CSS

JavaScript is a scripting or programming language that allows you to implement complex features on web pages

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

Internal CSS

<style>
CSS codes
</style>

External CSS

<link rel="stylesheet" type="text/css" href="test.css">

JavaScript is a very free-form language compared to Java. You do not have to declare all variables, classes, and methods. You do not have to be concerned with whether methods are public, private, or protected, and you do not have to implement interfaces. Variables, parameters, and function return types are not explicitly typed.

Internal JS

<script type = "text/javascript">
JS script codes
</script>

Extenal JS

<script type = "text/javascript" src="external.js"></script>

Head section is used as the first tag of any web page which signifies that it is beginning of web page. It has some elements that define header materials. It gives a descriptive title to a document.
